CriticalSection

The CriticalSection class simplifies the use of critical sections, otherwise known as process-local mutexes (in Windows a mutex is a global, inter-process synchronization object while critical section operates between threads of a single process). See Critical Section Objects for details.

The CriticalSection class has the following functions:

Function Related WinAPI function Description
CriticalSection InitializeCriticalSection Allocates and initializes critical section object.
~CriticalSection DeleteCriticalSection Destroys critical section object./td>
Enter EnterCriticalSection Enters critical section (locks the object).
Leave LeaveCriticalSection Leaves critical section (unlocks the object).
TryEnter TryEnterCriticalSection Enters critical section if it is not already entered by another thread.

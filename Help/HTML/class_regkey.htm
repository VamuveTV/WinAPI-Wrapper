WinAPI Wrapper

RegKey

The RegKey class simplifies access to and working with registry keys. See Registry Overview for details.

The RegKey class has the following functions:

Function Related WinAPI function Description
RegKey RegOpenKeyEx Either creates an empty key handle or opens a registry key using Open().
~RegKey RegCloseKey Calls Close().
Open RegOpenKeyEx Opens a registry key.
OpenUser RegOpenKeyEx Open a registry key within the HKEY_CURRENT_USER tree.
OpenMachine RegOpenKeyEx Open a registry key within the HKEY_LOCAL_MACHINE tree.
operator HKEY - Return the key handle.
Close RegCloseKey Closes registry key.
Create RegCreateKeyEx Creates a new registry key.
CreateUser RegCreateKeyEx Creates a new registry key within the HKEY_CURRENT_USER tree.
CreateMachine RegCreateKeyEx Creates a new registry key within the HKEY_LOCAL_MACHINE tree.
EnumKey RegEnumKeyEx Enumerates subkeys.
EnumKeys RegEnumKeyEx (STL only) Returns a vector with names of all subkeys.
QueryValueSize RegQueryValueEx Returns size (in bytes) of a value.
QueryValue RegQueryValueEx Retrieves a value from the registry.
SetValue RegSetValueEx Sets value.
QueryInfo RegQueryInfoKey Returns key information, such as number of subkeys, number of values, etc.
Delete RegDeleteKey Deletes a registry key.
DeleteKeyTree - Deletes a registry key with all its subkeys.
DeleteValue RegDeleteValue Deletes a value.
Flush RegFlushKey Flushes registry key.
Load RegLoadKey Loads a registry key from a file.
NotifyChangeValueAsync RegNotifyChangeKeyValue Watches for a value change and when it occurs, signals an event object.
NotifyChangeValue RegNotifyChangeKeyValue Waits for a value to change and then returns.
Replace RegReplaceKey Replaces the file backing a registry key and all its subkeys with another file.
Save RegSaveKey Saves registry key to a file.
Restore RegRestoreKey Restores a saved registry key.

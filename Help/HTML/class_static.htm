WinAPI Wrapper

Static

The Static class extends the Static standard Windows control with C++ functionality. It is derived from the Control class. See Static Controls Overview for details.

The Static class has the following functions:

Function Related message Description
Static - Creates the object optionally assigning an existing control handle to it.
Create CreateWindowEx Creates the control.
GetIcon STM_GETICON Returns the icon of a SS_ICON static.
GetBitmap STM_GETIMAGE Returns the bitmap associated with the static.
GetCursor STM_GETIMAGE Returns the cursor associated with the static.
SetIcon STM_SETICON Sets an icon to a SS_ICON static.
SetBitmap STM_SETIMAGE Sets a bitmap.
SetCursor STM_SETIMAGE Sets a cursor.

WinAPI Wrapper

CompatibleDC

The CompatibleDC class is used to contain (create and delete) a DC that is compatible with another DC (in the sense of device color space). It is derived from the DC class and has all its functionality. See DC class for details.

The CompatibleDC class has the following functions:

Function Related WinAPI function Description
CompatibleDC CreateCompatibleDC Creates the DC.
~CompatibleDC DeleteDC Destroys the DC.

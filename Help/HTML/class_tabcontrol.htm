TabControl

The TabControl class extends the TabControl standard Windows control with C++ functionality. It is derived from the Control class. See Tab Controls Overview for details.

The TabControl class has the following functions:

Function Related macro Description
TabControl - Creates the object optionally assigning an existing control handle to it.
Create CreateWindowEx Creates the control.
AdjustRect TabCtrl_AdjustRect Calculates adjusted rectangle taking into account tab control's stuff.
DeleteAllItems TabCtrl_DeleteAllItems Deletes all tabs.
DeleteItem TabCtrl_DeleteItem Deletes a tab.
DeselectAll TabCtrl_DeselectAll Deselects all tabs.
GetCurFocus TabCtrl_GetCurFocus Returns index of currently focused tab.
GetCurSel TabCtrl_GetCurSel Returns index of currently selected tab.
GetExtendedStyle TabCtrl_GetExtendedStyle Returns extended style.
GetImageList TabCtrl_GetImageList Returns image list.
GetItem TabCtrl_GetItem Returns tab configuration.
GetItemText TabCtrl_GetItem Returns tab text.
GetItemImage TabCtrl_GetItem Returns tab image index.
GetItemParam TabCtrl_GetItem Returns tab parameter.
GetItemState TabCtrl_GetItem Returns tab state.
GetItemCount TabCtrl_GetItemCount Returns number of tabs.
GetItemRect TabCtrl_GetItemRect Returns tab rectangle.
GetRowCount TabCtrl_GetRowCount Returns the number of rows of tabs.
GetToolTips TabCtrl_GetToolTips Returns ToolTip.
GetUnicodeFormat TabCtrl_GetUnicodeFormat Returns unicode format setting.
HighlightItem TabCtrl_HighlightItem Highlights a tab.
HitTest TabCtrl_HitTest Returns tab index at the specified screen coordinates.
InsertItem TabCtrl_InsertItem Inserts a new tab.
RemoveImage TabCtrl_RemoveImage Removes an image from the control's image list.
SetCurFocus TabCtrl_SetCurFocus Sets focus to a tab.
SetCurSel TabCtrl_SetCurSel Selects a tab.
SetExtendedStyle TabCtrl_SetExtendedStyle Sets extended style.
SetImageList TabCtrl_SetImageList Sets new image list.
SetItem TabCtrl_SetItem Alters settings of a tab.
SetItemState TabCtrl_SetItem Alters state of a tab.
SetItemExtra TabCtrl_SetItemExtra Changes the number of extra bytes per a tab.
SetItemSize TabCtrl_SetItemSize Resizes a tab.
SetMinTabWidth TabCtrl_SetMinTabWidth Sets minimum tab width.
SetPadding TabCtrl_SetPadding Sets padding around tabs' icons.
SetToolTips TabCtrl_SetToolTips Sets ToolTip.
SetUnicodeFormat TabCtrl_SetUnicodeFormat Sets unicode format setting.

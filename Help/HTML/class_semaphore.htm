WinAPI Wrapper

Semaphore

The Semaphore class simplifies the use of Semaphore Objects.

The Semaphore class has the following functions:

Function Related WinAPI function Description
Semaphore CreateSemaphore Creates semaphore object.
~Semaphore CloseHandle Destroys semaphore object.
Acquire WaitForSingleObject Acquires (decrements) semaphore. If the samphore equals to zero before the acquisition, the function waits until it becomes non-zero.
Release ReleaseSemaphore Releases (increments) semaphore.

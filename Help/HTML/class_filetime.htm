WinAPI Wrapper

FileTime

The FileTime class simplifies the use of time functions. It is derived from the public FILETIME structure.

The FileTime class has the following functions:

Function Related WinAPI function Description
FileTime SystemTimeToFileTime Constructs a time object, either an empty one or from FILETIME or from SYSTEMTIME.
operator FILETIME - Returns FILETIME structure.
operator SYSTEMTIME FileTimeToSystemTime Returns SYSTEMTIME structure (day, month, etc.).
operator= SystemTimeToFileTime Sets FILETIME or SYSTEMTIME structure.
ConvertToLocal FileTimeToLocalTime Assuming time is in UTC, converts it to local time.
ConvertToUTC LocalFileTimeToFileTime Assuming time is local time, converts it to UTC.
ObtainSystemTime GetSystemTimeAsFileTime Gets current time in UTC.
ObtainLocalTime GetLocalTime Gets current local time.
FormatTime GetTimeFormat Formats time (hour, minute, etc.) into a string. A static version of the functions works with SYSTEMTIME structure.
FormatDate GetDateFormat Formats date (day, month, etc.) into a string. A static version of the functions works with SYSTEMTIME structure.
operator== != < > <= >= CompareFileTime Compares two times.

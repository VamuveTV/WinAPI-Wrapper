ForegroundWindow

The ForegroundWindow class is an enhancement of the GenericWindow class. It initially contains the handle of the current foreground window in the system.

The ForegroundWindow class has the following functions:

Function Related WinAPI function Description
ForegroundWindow GetForegroundWindow Obtains handle of the current foreground window in the system.

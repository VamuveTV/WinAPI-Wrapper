WinAPI Wrapper

FileMapping

The FileMapping class in conjunction with the FileMapping::Map class simplifies the use of file mapping. See File Mapping Overview for details.

The FileMapping class has the following functions:

Function Related WinAPI function Description
FileMapping CreateFileMapping Assigns INVALID_HANDLE_VALUE or calls SetTo().
~FileMapping CloseHandle Calls Close().
operator HANDLE - Returns the file mapping handle.
SetTo CreateFileMapping Creates or opens file mapping.
IsOpen - Validates the handle.
Close CloseHandle Closes file mapping.

WinAPI Wrapper

ListBox

The ListBox class extends the ListBox standard Windows control with C++ functionality. It is derived from the Control class. See List Boxes Overview for details.

The ListBox class has the following functions:

Function Related message Description
ListBox - Creates the object optionally assigning an existing control handle to it.
Create CreateWindowEx Creates the control.
AddFile LB_ADDFILE Adds a filename to a listbox with directory listing.
AddString LB_ADDSTRING Adds a string.
InsertString LB_INSERTSTRING Inserts a string at a specified index.
DeleteString LB_DELETESTRING Deletes a string.
ResetContent LB_RESETCONTENT Removes all contents of the listbox.
FindString LB_FINDSTRING Finds an item begining with the speicified string.
FindStringExact LB_FINDSTRINGEXACT Finds a string.
Dir LB_DIR Fills the list box with a directory listing.
GetCount LB_GETCOUNT Returns the number of items.
GetCurSel LB_GETCURSEL Returns current selection.
SetCurSel LB_SETCURSEL Selects an item.
SelectString LB_SELECTSTRING Selects an item starting with the specified string.
GetHorizontalExtent LB_GETHORIZONTALEXTENT Returns number of pixels by which the list box can be scrolled horizontally.
SetHorizontalExtent LB_SETHORIZONTALEXTENT Sets number of pixels by which the list box can be scrolled horizontally.
GetItemData LB_GETITEMDATA Returns user item value.
SetItemData LB_SETITEMDATA Sets user item value.
GetItemHeight LB_GETITEMHEIGHT Returns height of every item.
SetItemHeight LB_SETITEMHEIGHT Sets height of every item.
GetTextLen LB_GETTEXTLEN Returns length of an item's string.
GetText LB_GETTEXT Returns an item's string.
GetLocale LB_GETLOCALE Returns current locale.
SetLocale LB_SETLOCALE Sets new locale.
GetTopIndex LB_GETTOPINDEX Returns index of the first visible item.
SetTopIndex LB_SETTOPINDEX Makes the specified item visible.
GetAnchorIndex LB_GETANCHORINDEX Returns index of an anchor item.
SetAnchorIndex LB_SETANCHORINDEX Makes an item an anchor item.
GetCaretIndex LB_GETCARETINDEX Returns index of an item that has the caret.
SetCaretIndex LB_SETCARETINDEX Sets the focus rectangle to a specific item.
GetItemRect LB_GETITEMRECT Returns the rectangle of an item.
GetSel LB_GETSEL Retrieves selection state of an item.
SetSel LB_SETSEL Alters selection state of an item.
GetSelCount LB_GETSELCOUNT Returns the number of selected items.
GetSelItems LB_GETSELITEMS Returns indices of selected items.
ItemFromPoint LB_ITEMFROMPOINT Returns index of an item at the specified display position in client coordinates.
SelItemRange LB_SELITEMRANGE Selects one or more consecutive items.
SetColumnWidth LB_SETCOLUMNWIDTH Sets column width in a multi-column listbox.
SetCount LB_SETCOUNT Sets the number of items.
SetTabStops LB_SETTABSTOPS Sets tab stops.
MakeDragList MakeDragList Changes a single-selection listbox into a drag listbox.

WinAPI Wrapper

Edit

The Edit class extends the Edit standard Windows control with C++ functionality. It is derived from the Control class. See Edit Controls Overview for details.

The Edit class has the following functions:

Function Related message Description
Edit - Creates the object optionally assigning an existing control handle to it.
Create CreateWindowEx Creates the control.
CanUndo EM_CANUNDO Determines whether there are any undoable actions.
CharFromPos EM_CHARFROMPOS Finds the closest character to a specified pixel.
EmptyUndoBuffer EM_EMPTYUNDOBUFFER Empties the undo buffer.
FormatLines EM_FMTLINES Sets whether the edit control should insert CR-CR-LF sequences on broken line ends.
GetFirstVisibleLine EM_GETFIRSTVISIBLELINE Retrieves the index of the first visible line.
GetHandle EM_GETHANDLE Returns memory handle of the edit buffer.
GetLimitText EM_GETLIMITTEXT Returns maximum number of characters that the user can enter.
GetLine EM_GETLINE Returns a line of text.
GetLineCount EM_GETLINECOUNT Returns number of lines in the edit box.
GetMargins EM_GETMARGINS Returns margin sizes.
GetModify EM_GETMODIFY Checks whether the contents of the edit were modified.
GetPasswordChar EM_GETPASSWORDCHAR Returns the current password character.
GetRect EM_GETRECT Returns the formatting rectangle.
GetSel EM_GETSEL Retrieves starting and ending positions of the currente selection.
GetThumb EM_GETTHUMB Returns the position of the scrollbox.
LimitText EM_LIMITTEXT Sets the maximum number of characters the user can enter.
LineFromChar EM_LINEFROMCHAR Returns line index of the line containing the character with the specified index.
LineIndex EM_LINEINDEX Returns character index of the first character in a line.
LineLength EM_LINELENGTH Returns the length of a line.
LineScroll EM_LINESCROLL Scrolls the text.
PosFromChar EM_POSFROMCHAR Returns client coordinates of a specified character (index).
ReplaceSel EM_REPLACESEL Replaces selection with a string.
Scroll EM_SCROLL Scrolls the text.
ScrollCaret EM_SCROLLCARET Scrolls until the caret is visible.
SetHandle EM_SETHANDLE Sets memory handle of the edit buffer.
SetLimitText EM_SETLIMITTEXT The same as LimitText.
SetMargins EM_SETMARGINS Sets margin sizes.
SetModify EM_SETMODIFY Sets the modify flag.
SetPasswordChar EM_SETPASSWORDCHAR Sets password character.
SetReadOnly EM_SETREADONLY Enables or disables writing.
SetRect EM_SETRECT Sets the formatting rectangle.
SetRectNP EM_SETRECTNP Same as SetRect, but does not redraw the control.
SetSel EM_SETSEL Selects a range of text.
SetTabStops EM_SETTABSTOPS Sets the tab stops.
Undo EM_UNDO Undoes last user modification.

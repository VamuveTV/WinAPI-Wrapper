WinAPI Wrapper

StockObject

The StockObject class obtains a GDI object from the stock of default GDI objects. The obtained object is indestructible.

The StockObject class has the following functions:

Function Related WinAPI function Description
StockObject GetStockObject
operator HBRUSH - Returns handle of a brush object.
operator HPEN - Returns handle of a pen object.
operator HFONT - Returns handle of a font object.
operator HPALETTE - Returns handle of a palette object.

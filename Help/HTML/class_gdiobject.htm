GDIObject

The GDIObject class is a C++ wrapper for HGDIOBJ GDI object handle. It is a base class for other classes that extend functionality of GDI objects.

The GDIObject class has the following functions:

Function Related WinAPI function Description
GDIObject - Obtains an existing HGDIOBJ handle or sets an empty handle.
IsValid - Checks whether the handle is valid (non-zero).
operator HGDIOBJ - Returns the GDI object handle.

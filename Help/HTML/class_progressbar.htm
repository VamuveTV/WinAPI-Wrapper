ProgressBar

The ProgressBar class extends the ProgressBar standard Windows control with C++ functionality. It is derived from the Control class. See Progress Bars Overview for details.

The ProgressBar class has the following functions:

Function Related message Description
ProgressBar - Creates the object optionally assigning an existing control handle to it.
Create CreateWindowEx Creates the control.
DeltaPos PBM_DELTAPOS Advances the current position by a specified amount.
GetPos PBM_GETPOS Returns current progress position.
GetRangeLow PBM_GETRANGE Returns low limit for allowable position.
GetRangeHigh PBM_GETRANGE Returns high limit for allowable position.
SetBarColor PBM_SETBARCOLOR Sets progress bar color.
SetBkColor PBM_SETBKCOLOR Sets progress background color.
SetPos PBM_SETPOS Sets progress position.
SetRange PBM_SETRANGE Sets allowable progress range.
SetStep PBM_SETSTEP Sets progress increment for StepIt.
StepIt PBM_STEPIT Increments progress by amount defined in SetStep.

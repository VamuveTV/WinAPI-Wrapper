WinAPI Wrapper

Font

The Font class provides automatic management (creating and destroying) of font GDI objects. It is derived from the GDIObjectBase class. See Fonts and Text Overview for details.

The Font class has the following functions:

Function Related WinAPI function Description
Font CreateFont CreateFontIndirect Creates a font object.
operator HFONT - Returns font handle.

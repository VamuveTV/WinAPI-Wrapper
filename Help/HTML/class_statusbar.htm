WinAPI Wrapper

StatusBar

The StatusBar class extends the StatusBar standard Windows control with C++ functionality. It is derived from the Control class. See Status Bars Overview for details.

The StatusBar class has the following functions:

Function Related message Description
StatusBar - Creates the object optionally assigning an existing control handle to it.
Create CreateWindowEx Creates the control.
GetBorders SB_GETBORDERS Returns status bar border sizes.
GetIcon SB_GETICON Returns part's icon.
GetParts SB_GETPARTS Returns number of parts and where they end.
GetRect SB_GETRECT Returns part's rectangle.
GetPartType SB_GETTEXTLENGTH Returns part type.
GetTextLen SB_GETTEXTLENGTH Returns string length of a part.
GetText SB_GETTEXT Returns part's string.
GetTipText SB_GETTIPTEXT Returns part's tip text.
GetUnicodeFormat SB_GETUNICODEFORMAT Returns unicode format setting.
IsSimple SB_ISSIMPLE Tests whether the status bar is in simple mode.
SetBkColor SB_SETBKCOLOR Sets background color.
SetIcon SB_GETICON Sets part's icon.
SetMinHeight SB_SETMINHEIGHT Sets minimum height of drawing area.
SetParts SB_SETPARTS Sets part sizes.
SetText SB_SETTEXT Sets part's text.
SetTipText SB_SETTIPTEXT Sets part's tip text.
SetUnicodeFormat SB_SETUNICODEFORMAT Sets unicode format setting.
Simple SB_SIMPLE Switches between simple mode and displaying all parts.

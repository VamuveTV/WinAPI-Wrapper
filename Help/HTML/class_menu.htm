WinAPI Wrapper

Menu

The Menu class extends a HMENU object with C++ functionality. See Menus Overview for details.

The Menu class has the following functions:

Function Related WinAPI function Description
Menu - Creates an empty object or obtains a menu handle.
operator HMENU - Returns the menu handle.
Load LoadMenu Loads menu from a resource.
AppendGeneric AppendMenu Appends a menu in a generic way.
Append AppendMenu Appends a menu either with a string or with a bitmap. Another version of this function can also append popup menus.
AppendSeparator AppendMenu Appends a separator menu item.
CheckItem CheckMenuItem Checks or unchecks a "check"-type menu item.
CheckRadioItem CheckMenuRadioItem Checks a "radio"-type menu item in a given group.
Create CreateMenu Creates a new menu.
CreatePopupMenu CreatePopupMenu Creates a new popup menu.
Delete DeleteMenu Deletes a menu item.
Destroy DestroyMenu Destroys menu object.
EnableItem EnableMenuItem Enables or disables a menu item.
GetDefaultItem GetMenuDefaultItem Retrieves default menu item.
GetInfo GetMenuInfo Retrieves a MENUINFO structure.
GetItemCount GetMenuItemCount Returns the number of menu items.
GetItemID GetMenuItemID Retrieves command identifier of a menu item.
GetItemInfo GetMenuItemInfo. Gets a MENUITEMINFO structure.
GetState GetMenuState Retrieves state of a menu item.
GetString GetMenuString Returns menu item string.
GetSubMenu GetSubMenu Returns a sub-menu at a specific item.
InsertGeneric InsertMenu Inserts a generic menu item.
Insert InsertMenu Inserts a menu item either with a string or with a bitmap. Another version of this function can also append popup menus.
InsertSeparator InsertMenu Inserts a separator menu item.
IsMenu IsMenu Checks whether the objects contains a valid menu handle to an existing menu object.
Remove RemoveMenu Removes a menu item or detaches a sub-menu.
SetDefaultItem SetMenuDefaultItem Sets default menu item.
SetInfo SetMenuInfo Sets a MENUINFO structure.
SetItemBitmaps SetMenuItemBitmaps Sets checked and unchecked state bitmaps.
SetItemInfo SetMenuItemInfo Sets a MENUITEMINFO structure.
TrackPopupMenu TrackPopupMenu TrackPopupMenuEx Executes a popupmenu loop.

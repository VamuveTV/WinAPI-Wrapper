WinAPI Wrapper

GetFileNameDialog

The GetFileNameDialog class is an extension of the OPENFILENAME structure and it is derived from the structure. The common dialog box created by this class gets the user to select one or more files. For obtaining a folder name from the user use GetFolderDialog class.

The GetFileNameDialog class has the following functions:

Function Related WinAPI function Description
GetFileNameDialog - Constructs the OPENFILENAME structure.
GoOpen GetOpenFileName Opens the "Open" dialog.
GoSave GetSaveFileName Opens the "Save" dialog.
SetFilterIndex Sets initial index of the file type filter.
GetFilterIndex - Determines which filter the user chosen after GoXXX() function has been called.
SetFileName - Sets initial file name.
GetPathName - Gets whole path to the file selected by the user.
GetFileName - Gets name of the file selected by the user.
GetExtension - Gets extension of the file selected by the user.
GetFileNameOffset - Gets offset of the file name in the file path.
GetExtensionOffset - Gets offset of the extension in the file path.

WinAPI Wrapper

Address

The Address class is a helper class for sockets. It is an IP address in conjunction with a port number.

The Address class has the following functions:

Function Related WinAPI function Description
Address gethostbyname Creates address object from IP and port, from name or from sockaddr.
operator== - Compares two addresses for equality.
operator!= - Compares two addresses for inequality.
SetPort - Sets new port.
GetAddress - Returns in_addr.
GetPort - Returns port number.
GetSockAddr - Returns sockaddr structure.
GetName gethostbyaddr Gets address name string.

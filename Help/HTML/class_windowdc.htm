WindowDC

The WindowDC class is used to operate on a window DC, including the Window's non-client area. It is derived from the DC class and has all its functionality. See DC class for details.

The WindowDC class has the following functions:

Function Related WinAPI function Description
WindowDC GetWindowDC Obtains the DC for drawing.
~WindowDC ReleaseDC Releases the DC after drawing with it.

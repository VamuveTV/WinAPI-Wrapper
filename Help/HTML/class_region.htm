Region

The Region class provides automatic management (creating and destroying) of region GDI objects. It is derived from the GDIObjectBase class. See Regions Overview for details.

The Region class has the following functions:

Function Related WinAPI function Description
Region CreateRectRgnIndirect CreatePolygonRgn ExtCreateRegion Creates a region.
operator HRGN - Returns the region handle.
SetRect SetRectRgn Converts region to a rectangular region of the specified dimensions.
IsPtInside PtInRegion Determines whether a point is inside the region.
GetData GetRegionData Obtains region description in form of a RGNDATA structure.
OffsetBy OffsetRgn Moves region by the specified amounts in X and Y directions.
GetBox GetRgnBox Returns a bounding rectangle of the region.

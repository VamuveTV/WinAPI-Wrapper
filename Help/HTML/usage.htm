WinAPI Wrapper

Usage guide

The following topics describe, step by step, how to use the WinAPI Wrapper in certain situations.

WinAPI Wrapper installation notes
Discusses how to correctly install the WinAPI Wrapper. This topic contains basic knowledge of using static libraries in C(++).
Creating a project (Microsoft Visual Studio only)
Pictures out important steps in creating a working project that takes advantage of the WinAPI Wrapper library in Microsoft Visual Studio environment.
Creating windows and controls
Describes how to create your own windows and controls. It also shows the differences between your own controls and the stadard controls' wrapper classes provided by the WinAPI Wrapper library.
Creating dialog boxes
Describes how to create modal and modeless dialog boxes.
Accessing controls in dialog boxes
Shows how to access controls in dialog boxes.
The application object
Discusses inner working of an application created with the default WinMain function provided by the WinAPI Wrapper library. Also shows how to use accelerators.
MDI applications
Shows how to create MDI applications with the WinAPI Wrapper.
Examples
Shows example applications that are based on different main window types.
Using custom window classes
Describes how to create and use custom window classes.
Miscellaneous wrapper classes
Describes miscallenous WinAPI Wrapper classes and show how to use them.
GDI wrapper classes
Shows in examples how to use the GDI wrapper classes.
Multithreading
Discusses the simplifications of multithreading provided by the WinAPI Wrapper library.
COM interface pointers
Introduces versatile COM interface safe pointer.
Pure wrapper
Shows how to get rid of additional methods in Window-derived classes.

WinAPI Wrapper

Pager

The Pager class extends the Pager standard Windows control with C++ functionality. It is derived from the Control class. See Pagers Overview for details.

The Pager class has the following functions:

Function Related message Description
Pager - Creates the object optionally assigning an existing control handle to it.
Create CreateWindowEx Creates the control.
ForwardMouse PGM_FORWARDMOUSE Enables or disables mouse forwarding.
GetBkColor PGM_GETBKCOLOR Returns background color.
GetBorder PGM_GETBORDER Returns border dimensions.
GetButtonSize PGM_GETBUTTONSIZE Returns buttons size.
GetButtonState PGM_GETBUTTONSTATE Returns current state of buttons.
GetPos PGM_GETPOS Returns current child scroll position.
RecalcSize PGM_RECALCSIZE Recalculates size.
SetBkColor PGM_SETBKCOLOR Sets background color.
SetBorder PGM_SETBORDER Sets border dimensions.
SetButtonSize PGM_SETBUTTONSIZE Sets buttons size.
SetChild PGM_SETCHILD Sets new child.
SetPos PGM_SETPOS Sets new child scroll position.

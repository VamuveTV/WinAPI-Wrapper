WinAPI Wrapper

Rebar

The Rebar class extends the Rebar standard Windows control with C++ functionality. It is derived from the Control class. See Rebars Overview for details.

The Rebar class has the following functions:

Function Related message Description
Rebar - Creates the object optionally assigning an existing control handle to it.
Create CreateWindowEx Creates the control.
BeginDrag RB_BEGINDRAG Begins dragging a band.
DeleteBand RB_DELETEBAND Deletes a band.
DragMove RB_DRAGMOVE Moves a band during dragging.
EndDrag RB_ENDDRAG Ends dragging a band.
GetBandBorders RB_GETBANDBORDERS Returns band borders.
GetBandCount RB_GETBANDCOUNT Returns the number of bands.
GetBandInfo RB_GETBANDINFO Returns band configuration.
GetBarHeight RB_GETBARHEIGHT Returns the height of the rebar.
GetBarInfo RB_GETBARINFO Returns rebar's image list.
GetBkColor RB_GETBKCOLOR Returns rebar's background color.
GetColorScheme RB_GETCOLORSCHEME Returns color scheme.
GetPalette RB_GETPALETTE Returns palette.
GetRect RB_GETRECT Returns band rectangle.
GetRowCount RB_GETROWCOUNT Returns current number of rows.
GetRowHeight RB_GETROWHEIGHT Returns height of a row.
GetTextColor RB_GETTEXTCOLOR Returns text color.
GetToolTips RB_GETTOOLTIPS Returns ToolTip.
GetUnicodeFormat RB_GETUNICODEFORMAT Returns unicode format setting.
HitTest RB_HITTEST Returns band under the specified screen coordinates.
IDToIndex RB_IDTOINDEX Translates band ID into band index.
InsertBand RB_INSERTBAND Inserts a new band.
MaximizeBand RB_MAXIMIZEBAND Maximizes band.
MinimizeBand RB_MINIMIZEBAND Minimizes band.
MoveBand RB_MOVEBAND Moves band to a new location.
SetBandInfo RB_SETBANDINFO Sets band configuration.
SetBarInfo RB_SETBARINFO Sets new image list.
SetBkColor RB_SETBKCOLOR Sets rebar's background color.
SetColorScheme RB_SETCOLORSCHEME Sets color scheme.
SetPalette RB_GETPALETTE Sets palette.
SetParent RB_SETPARENT Sets parent (notification destintation).
SetTextColor RB_GETTEXTCOLOR Sets text color.
SetToolTips RB_SETTOOLTIPS Sets ToolTip.
SetUnicodeFormat RB_SETUNICODEFORMAT Sets unicode format setting.
ShowBand RB_SHOWBAND Shows or hides a band.
SizeToRect RB_SIZETORECT Calculates layout for the given set of bands.

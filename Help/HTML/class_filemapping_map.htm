FileMapping::Map

The FileMapping::Map represents an area of memory that is a mapped file mapping. It is an utility class for the FileMapping. See File Mapping Overview for details.

The FileMapping::Map class has the following functions:

Function Related WinAPI function Description
FileMapping::Map MapViewOfFile Maps a view of file.
FileMapping::~Map UnmapViewOfFile Unmaps a view of file.
operator void* - Returns starting address of the mapped buffer.
operator char* - Returns starting address of the mapped buffer.
operator[] - Provides direct array access to the mapped buffer. (No offest validity check is performed)
GetSize - Returns size of the mapped area.
Size - Returns size of the mapped area.

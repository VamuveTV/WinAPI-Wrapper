CheckBox

The CheckBox class is a special case of a button - a checkbox. It is derived from the Button class and has all its functionality.

The CheckBox class has the following functions:

Function Related message Description
CheckBox - Creates the object optionally assigning an existing control handle to it.
Create CreateWindowEx Creates the control.

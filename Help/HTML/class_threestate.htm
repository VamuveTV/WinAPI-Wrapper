ThreeState

The ThreeState class is a special case of a button - a three-state checkbox. It is derived from the Button class and has all its functionality.

The ThreeState class has the following functions:

Function Related message Description
ThreeState - Creates the object optionally assigning an existing control handle to it.
Create CreateWindowEx Creates the control.

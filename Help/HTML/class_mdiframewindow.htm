MDIFrameWindow

The MDIFrameWindow class is a Window-derived class and is meant for MDI frame windows. The only difference between it and Window is that it has a few additional MDIXXX functions and that you must call CreateClient during processing OnCreate in order to create the MDI client window.

The MDIFrameWindow class has the following functions:

Function Related WinAPI phenomena Description
CreateClient - Creates the MDI client window. This function must be called when processing the WM_CREATE message.
GetClientWindow - Returns the MDI client window.
MDIActivate WM_MDIACTIVATE Activates the specified MDI child window.
MDICascade WM_MDICASCADE Cascades all MDI child windows.
MDICreate WM_MDICREATE Creates an MDI child window.
MDIDestroy WM_MDIDESTROY Destroys an MDI child window.
MDIGetActive WM_MDIGETACTIVE Retruns the active MDI child window.
MDIIconArrange WM_MDIICONARRANGE Arranges all minimized MDI child windows.
MDIMaximize WM_MDIMAXIMIZE Maximizes an MDI child window.
MDINext WM_MDINEXT Activates the next MDI child window.
MDIPrevious WM_MDINEXT Activates the previous MDI child window.
MDIRefreshMenu WM_MDIREFRESHMENU Applies changes to the MDI window menu.
MDIRestore WM_MDIRESTORE Restores a minimized or maximized MDI child window.
MDISetMenu WM_MDISETMENU Sets new MDI window menu.
MDITile WM_MDITILE Tiles all MDI child windows.

WinAPI Wrapper

GenericWindow

The GenericWindow class extends HWND window handle with C++ functionality. See Windows Overview for details.

The GenericWindow class has the following functions:

Function Related WinAPI function Description
GenericWindow - Constructs an empty object with no valid window handle or obtains an existing window handle.
operator HWND - Returns window handle.
Create CreateWindowEx Creates window.
Show ShowWindow Shows or hides window.
ShowAsync ShowWindowAsync Show or hides a window on another thread.
IsVisible IsWindowVisible Checks whether the window is visible.
Update UpdateWindow Repaints dirty areas of a window.
Redraw RedrawWindow Redraws specified areas of window.
Invalidate InvalidateRect Makes whole client area dirty.
InvalidateRect InvalidateRect Makes a specified rectangle on client area dirty.
InvalidateRgn InvalidateRgn Makes a region of client area dirty.
ValidateRect ValidateRect Makes a rectangle on client area non-dirty.
ValidateRgn ValidateRgn Makes a region of client area non-dirty.
Destroy DestroyWindow Destroys the window.
BringToTop BringWindowToTop Makes the window the topmost window.
SetForeground SetForegroundWindow Makes the window a foreground window.
ChildWindowFromPoint ChildWindowFromPoint Finds the first child window that intersects the specified point.
GetLastActivePopup GetLastActivePopup Returns the last active popup window.
GetNext GetWindow Returns next sibling of the window.
GetPrev GetWindow Returns previous sibling of the window.
GetFirst GetWindow Returns first sibling of the window.
GetLast GetWindow Returns last sibling of the window.
GetChild GetWindow Returns topmost sibling of the window.
GetTop GetTopWindow Returns topmost child of the window.
GetParent GetParent Returns parent of the window.
SetParent SetParent Sets parent of the window.
Close CloseWindow Minimizes window. (Mislieading function name!)
IsIconic IsIconic Checks if the window is minimized.
Open OpenIcon Restores a minimized window.
Enable EnableWindow Enables or disables window.
IsEnabled IsWindowEnabled Checks whether the window is enabled.
ShowOwnedPopups ShowOwnedPopups Shows or hides all popups.
GetClientRect GetClientRect Returns the rectangle of client area (in client coordinates).
GetWindowRect GetWindowRect Returns the rectangle of the whole window in screen coordinates.
GetWindowRectOnParent GetWindowRect Returns the rectangle of the whole window in its parent's client coordinates.
IsChild IsChild Checks whether another window is the child of the window.
IsWindow IsWindow Checks whether the handle is a valid window handle.
IsZoomed IsZoomed Checks whether the window is maximized.
GetPlacement GetWindowPlacement Returns the window placement in form of WINDOWPLACEMENT structure.
SetPlacement SetWindowPlacement Sets the window placement in form of WINDOWPLACEMENT structure.
GetTextLength GetWindowTextLength Returns the length of the window title string.
GetText GetWindowText Returns window title.
SetText SetWindowText Sets window title.
Move MoveWindow Changes position and dimensions of the window.
SetPos SetWindowPos Changes position, dimensions or Z-order of the window.
PostMessage PostMessage Puts a message in the window thread's queue, but does not execute it immediately.
SendMessage SendMessage Invokes window procedure in order to process the specified message.
SendMessageTimeout SendMessageTimeout The same as SendMessage, but if the window is on another thread the function waits as long as the timeout lets it to.
MessageBox MessageBox Displayes a message box as the child of the window.
SendDlgItemMessage SendDlgItemMessage Calls SendMessage for a control with the specified identifier.
GetDlgCtrlID GetDlgCtrlID Retrieves the identifier of the window (assuming it is a control).
GetDlgItem GetDlgItem Gets window handle of a specific control.
GetNextDlgGroupItem GetNextDlgGroupItem Gets the next control in the group.
GetPrevDlgGroupItem GetNextDlgGroupItem Gets the previous control in the group.
GetNextDlgTabItem GetNextDlgTabItem Gets the next control that can be navigated with TAB.
GetPrevDlgTabItem GetNextDlgTabItem Gets the previous control that can be navigated with TAB.
GetDlgItemInt GetDlgItemInt Gets value in a control as a signed integer.
GetDlgItemUInt GetDlgItemInt Gets value in a control as an unsigned integer.
SetDlgItemInt SetDlgItemInt Sets an integer value to a control (either signed or unsigned).
GetDlgItemText GetDlgItemText Retrieves text from a control.
SetDlgItemText SetDlgItemText Sets text to a control.
MapDialogRect MapDialogRect Converts a rectangle from dialog base units to screen units.
CheckDlgButton CheckDlgButton Checks or unchecks a button (e.g. a checkbox).
CheckRadioButton CheckRadioButton Checks or unchecks a button (e.g. a radio button).
IsDlgButtonChecked IsDlgButtonChecked Checks whether a button is checked.
EnableScrollBar EnableScrollBar Enables or disabled window scrollbars.
GetScrollInfo GetScrollInfo Retrieves parameters of a scrollbar in form of a SCROLLINFO structure.
SetScrollInfo SetScrollInfo Sets parameters of a scrollbar in form of a SCROLLINFO structure.
ShowScrollBar ShowScrollBar Shows or hides a window scrollbar.
Scroll ScrollWindow Scrolls contents of window's client area.
SetFont WM_SETFONT Sends the WM_SETFONT message to set the window font.
GetFont WM_GETFONT Sends the WM_GETFONT message in order to retrieve the window font.
SetIcon WM_SETICON Sends the WM_SETICON message to set the window icon.
GetIcon WM_GETICON Retrieves window icon throught the WM_GETICON message.
ScreenToClient ScreenToClient Converts a point or a rectangle from screen to client coordinates.
ClientToScreen ClientToScreen Converts a point or a rectangle from client to window coordinates.
SetFocus SetFocus Sets keyboard focus to the window.
SetActive SetActiveWindow Makes the window active.
SetCapture SetCapture Makes the window capture all mouse input.
GetMenu GetMenu Returns window's main menu.
DrawMenuBar DrawMenuBar Redraws the window's main menu.
GetSystemMenu GetSystemMenu Gets the window's system menu.
SetMenu SetMenu Assigns a new main menu to the window.
DrawAnimatedRects DrawAnimatedRects Draws an animation of a window being maximized or minimized.
GetObject<T> - Obtains pointer to a window object associated with the window. Use this template function very carefully when you are absolutely sure that you have a handle to a Window-derived object.

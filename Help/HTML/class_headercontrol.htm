WinAPI Wrapper

HeaderControl

The HeaderControl class extends the HeaderControl standard Windows control with C++ functionality. It is derived from the Control class. See Header Controls Overview for details.

The HeaderControl class has the following functions:

Function Related macro Description
HeaderControl - Creates the object optionally assigning an existing control handle to it.
Create CreateWindowEx Creates the control.
CreateDragImage Header_CreateDragImage Creates an image for drag operation.
DeleteItem Header_DeleteItem Deletes an item.
GetImageList Header_GetImageList Returns the header's image list.
GetItem Header_GetItem Returns item data in form of the HDITEM structure.
GetItemCount Header_GetItemCount Returns number of items.
GetItemRect Header_GetItemRect Returns rectangle containing the item.
GetOrderArray Header_GetOrderArray Returns order of items.
GetUnicodeFormat Header_GetUnicodeFormat Returns unicode format setting.
InsertItem Header_InsertItem Inserts a new item.
Layout Header_Layout Returns size and position of an item.
OrderToIndex Header_OrderToIndex Retrieves item index based on its order position.
SetHotDivider Header_SetHotDivider Makes an item look like the drop target.
SetImageList Header_SetImageList Sets new image list.
SetItem Header_SetItem Sets item data in form of the HDITEM structure.
SetItemText Header_SetItem Sets item string.
SetItemFormat Header_SetItem Sets item format.
SetOrderArray Header_SetOrderArray Sets order of items.
SetUnicodeFormat Header_SetUnicodeFormat Sets unicode format setting.

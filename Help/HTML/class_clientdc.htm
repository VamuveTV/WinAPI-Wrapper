WinAPI Wrapper

ClientDC

The ClientDC class is used to operate on DCs limited to windows' client areas. It is derived from the DC class and has all its functionality. See DC class for details.

The ClientDC class has the following functions:

Function Related WinAPI function Description
ClientDC GetDC Obtains the DC for drawing.
~ClientDC ReleaseDC Releases the DC after drawing with it.

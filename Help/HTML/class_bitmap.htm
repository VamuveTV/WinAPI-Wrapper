WinAPI Wrapper

Bitmap

The Bitmap class provides automatic management (creating and destroying) of bitmap GDI objects. It is derived from the GDIObjectBase class. See Bitmaps Overview for details.

The Bitmap class has the following functions:

Function Related WinAPI function Description
Bitmap CreateBitmap CreateCompatibleBitmap CreateDIBitmap Creates a new bitmap.
operator HBITMAP - Returns bitmap handle.
SetDIBits SetDIBits Sets bits to the bitmap.
GetDIBits GetDIBits Gets bits of the bitmap.
Load LoadBitmap Loads the bitmap from a resource.

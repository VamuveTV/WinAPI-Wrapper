WinAPI Wrapper

ToolTip

The ToolTip class extends the ToolTip standard Windows control with C++ functionality. It is derived from the Control class. See ToolTips Overview for details.

The ToolTip class has the following functions:

Function Related message Description
ToolTip - Creates the object optionally assigning an existing control handle to it.
Create CreateWindowEx Creates the control.
Activate TTM_ACTIVATE Activates or deactivates the tooltip.
AddTool TTM_ADDTOOL Adds a tool/control for which a tooltip will be displayed.
DelTool TTM_DELTOOL Removes a tool from the tooltips list.
EnumTools TTM_ENUMTOOLS Returns information about a tool.
GetCurrentTool TTM_GETCURRENTTOOL Returns the currently used tool.
GetMargin TTM_GETMARGIN Returns the tooltip window margins.
GetMaxTipWidth TTM_GETMAXTIPWIDTH Returns maximum tooltip window width.
GetTipBkColor TTM_GETTIPBKCOLOR Returns tooltip window background color.
GetTipTextColor TTM_GETTIPTEXTCOLOR Returns tooltip window text color.
GetToolCount TTM_GETTOOLCOUNT Returns number of registered tools.
GetToolInfo TTM_GETTOOLINFO Returns tool configuration.
HitTest TTM_HITTEST Returns tool under specified screen coordinates.
NewToolRect TTM_NEWTOOLRECT Sets a new bounding rectangle for a tool.
Pop TTM_POP Destroys displayed tooltip window.
RelayEvent TTM_RELAYEVENT Passes a mouse message to the tooltip.
SetDelayTime TTM_SETDELAYTIME Set tooltip window times.
SetMargin TTM_SETMARGIN Sets tooltip window margins.
SetMaxTipWidth TTM_SETMAXTIPWIDTH Sets maximum tooltip window width.
SetTipBkColor TTM_SETTIPBKCOLOR Sets tooltip window background color.
SetTipTextColor TTM_SETTIPTEXTCOLOR Sets tooltip window text color.
SetTitle TTM_SETTITLE Adds an icon and title to the tooltip.
SetToolInfo TTM_SETTOOLINFO Sets tool configuration.
TrackActivate TTM_TRACKACTIVATE Activates or deactivates a tracking tooltip.
TrackPosition TTM_TRACKPOSITION Sets position of a tracking tooltip.
Update TTM_UPDATE Redraws current tool.
UpdateTipText TTM_UPDATETIPTEXT Sets new tip text for a tool.
WindowFromPoint TTM_WINDOWFROMPOINT Allows to display tooltip for another point.

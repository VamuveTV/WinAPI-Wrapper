WinAPI Wrapper

Trackbar

The Trackbar class extends the Trackbar standard Windows control with C++ functionality. It is derived from the Control class. See Trackbars Overview for details.

The Trackbar class has the following functions:

Function Related message Description
Trackbar - Creates the object optionally assigning an existing control handle to it.
Create CreateWindowEx Creates the control.
GetPos TBM_GETPOS Returns current position.
GetRangeMin TBM_GETRANGEMIN Returns the lower limit of allowable position.
GetRangeMax TBM_GETRANGEMAX Returns the upper limit of allowable position.
SetPos TBM_SETPOS Sets trackbar position.
SetRange TBM_SETRANGEMIN TBM_SETRANGEMAX Sets allowable range of trackbar positions.
ClearSel TBM_CLEARSEL Clears current selection range.
ClearTics TBM_CLEARTICS Removes tick marks.
GetChannelRect TBM_GETCHANNELRECT Returns rectangle for trackbar's channel.
GetLineSize TBM_GETLINESIZE Returns number of positions the trackbar moves in response to keyboard arrows.
SetLineSize TBM_SETLINESIZE Sets number of positions the trackbar moves in response to keyboard arrows.
GetNumTics TBM_GETNUMTICS Returns number of tics.
GetPageSize TBM_GETPAGESIZE Returns number of positions the trackbar moves in response to keyboard page up and down.
SetPageSize TBM_SETPAGESIZE Sets number of positions the trackbar moves in response to keyboard page up and down.
GetPTics TBM_GETPTICS Retrieves a pointer to an array with positions of tics.
GetSelStart TBM_GETSELSTART Returns the beginning of selection.
GetSelEnd TBM_GETSELEND Returns the end of selection.
GetThumbLength TBM_GETTHUMBLENGTH Returns size of the thumb.
GetThumbRect TBM_GETTHUMBRECT Returns the rectangle that contains the thumb.
GetTic TBM_GETTIC Returns position of a specific tick mark.
GetTicPos TBM_GETTICPOS Returns screen position of a specific tick mark.
SetTic TBM_SETTIC Sets a tick mark at a specific position.
SetTicFreq TBM_SETTICFREQ Sets tick marks with a specified frequency.
GetToolTips TBM_GETTOOLTIPS Returns the tool tip control.
GetUnicodeFormat TBM_GETUNICODEFORMAT Returns unicode format setting.
SetLeftBuddy TBM_SETBUDDY Sets the left buddy window.
SetRightBuddy TBM_SETBUDDY Sets the right buddy window.
GetLeftBuddy TBM_GETBUDDY Returns the left buddy window.
GetRightBuddy TBM_GETBUDDY Returns the right buddy window.
SetSel TBM_SETSELSTART TBM_SETSELEND Selects a range on the trackbar.
SetThumbLength TBM_SETTHUMBLENGTH Sets thumb size.
SetTipSide TBM_SETTIPSIDE Positions a tooptip control.
SetToolTips TBM_SETTOOLTIPS Sets tooltip.
SetUnicodeFormat TBM_SETUNICODEFORMAT Sets unicode format setting.

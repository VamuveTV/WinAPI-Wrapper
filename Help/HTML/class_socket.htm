WinAPI Wrapper

Socket

The Socket class simplifies the use of BSD sockets (Windows Sockets 2).

The Socket class has the following functions:

Function Related WinAPI function Description
Socket socket Creates an empty socket, a socket object from an existing socket handle or a new socket using socket function.
operator= - Moves socket handle from one object to another.
~Socket shutdown closesocket Calls Close()
IsValid - Checks whether the socket handle is valid.
GetError errno Returns error code after the last socket operation failed.
Bind bind Binds an address to the socket.
Connect connect Makes connection to a specified address/port pair.
GetPeerName getpeername Returns address (name) to which the socket is connected.
GetSockName getsockname Returns socket's address (name).
Listen listen Waits for an incoming connection.
Accept accept Accepts an incoming connection making a new socket with the connection.
Select select Waits until there is anything to read from the socket (any data has been received).
Send send Sends data over a connection.
Receive recv Receives data sent over a connection.
SendTo sendto Sends data to a specified address/port without a connection,
ReceiveFrom recvfrom Receives data on a nonconnected socket.
SetBlocking WSAIoctl Puts socket into a blocking or nonblocking state.
Close shutdown closesocket Shuts down a connection and destroys the socket.

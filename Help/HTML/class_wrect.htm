WinAPI Wrapper

WRect

The WRect class extends the RECT structure with C++ functionality.

The WRect class has the following functions:

Function Related WinAPI function Description
WRect - Builds WRect in miscellaneous ways.
operator== - Tests two rectangles for equality.
operator!= - Tests two rectangles for inequality.
operator& - Calculates intersection of two rectangles.
operator| - Calculates sum of two rectangles.
Width - Calculates width of the rectangle.
Height - Calculates height of the rectangle.
Size - Calculates size of the rectangle (in form of WSize).
GetSize - Calculates dimensions of the rectangle (in form of WSize).
SetSize - Sets new dimensions of the rectangle.
Contains - Tests whether the rectangle contains point or another rectangle entirely.
IsValid - Checks whether the rectangle is valid (has positive dimensions).
Intersects - Tests whether the rectangle intersects another rectangle.
LeftTop - Returns left top corner as WPoint.
LeftBottom - Returns left bottom corner as WPoint.
RightTop - Returns right top corner as WPoint.
RightBottom - Returns right bottom corner as WPoint.
InsetBy - Shrinks the rectangle by given amounts in X and Y axes.
OffsetBy - Moves the rectangle by a given amount (adding X and Y values to respective borders).
OffsetTo - Moves the rectangle to a given left-top position.
InsetBySelf - These functions do the same job as the above functions, but they also return reference to the rectangle itself.
OffsetBySelf
OffsetToSelf
InsetByCopy - These functions do the same job as the above functions, but they don't alter the actual rectangle, but return an altered copy of it.
OffsetByCopy
OffsetToCopy

Several functions in this class have been inspired by similar functionality of the Rect class in the BeOS API.

WinAPI Wrapper

WPoint

The WPoint class extends the POINT structure with C++ functionality.

The WPoint class has the following functions:

Function Related WinAPI function Description
WPoint -
~WPoint -
operator+ - Adds fields of two points parallely.
operator- - Subtracts fields of two points parallely.
operator- (unary) - Calculates negative fields of the point.
operator+= - Adds fields of another point parallely.
operator-= - Subtracts fields of another point parallely.
operator== - Compares two points for equality.
operator!= - Compares two points for inequality.

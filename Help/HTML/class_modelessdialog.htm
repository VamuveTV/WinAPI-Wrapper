ModelessDialog

The ModelessDialog class is something between Window and Dialog. Although it is derived from Dialog and receives the same messages as Dialog, it is created and destroyed the same way Window is (created dynamically with the new operator and with calling Create afterwards and destroyed automatically after Destroy is called).

The ModelessDialog class has the following functions:

Function Related WinAPI function Description
Create CreateDialog Creates the dialog box.

Remarks

Although the Go and EndDialog functions from Dialog are not currently hidden, you should not call them in ModelessDialog-derived classes.

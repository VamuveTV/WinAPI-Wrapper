WindowClass

The WindowClass class is a required component for the Window class. It serves the sole purpose for registering a window class with RegisterClass function. It is neccessary to use this class in order for Window to work correctly.

The WindowClass class has the following functions:

Function Related WinAPI function Description
WindowClass RegisterClass Registers the window class. The protected version is for derived classes that supply their own window procedure.
operator LPCTSTR - Returns name of the registered window class.

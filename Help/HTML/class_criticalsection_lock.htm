WinAPI Wrapper

CriticalSection::Lock

The CriticalSection::Lock class enters (locks) a critical section object for the lifetime of a lock object.

The CriticalSection::Lock class has the following functions:

Function Related WinAPI function Description
CriticalSection::Lock EnterCriticalSection Calls Enter() on a CriticalSection object.
CriticalSection::~Lock LeaveCriticalSection Calls Leave() on a CriticalSection object.

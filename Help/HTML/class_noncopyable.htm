NonCopyAble

The NonCopyAble class is a helper class. It is used as a base class for other classes to prevent them being copied. You cannot create objects of NonCopyAble class, because it has its constructor and destructor private. You cannot copy objects derived from NonCopyAble class, because it has its copy constructor and copy (assignment) operator= private.

Certain objects cannot be copied, because either the WinAPI makes it impossible to copy them or it would be too complex to copy them. Hence these objects' classes are derived from NonCopyAble.

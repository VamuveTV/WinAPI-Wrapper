WSize

The WSize class extends the SIZE structure with C++ functionality.

The WSize class has the following functions:

Function Related WinAPI function Description
WSize -
~WSize -
operator+ - Adds fields of two sizes parallely.
operator- - Subtracts fields of two sizes parallely.
operator- (unary) - Calculates negative fields of the size.
operator+= - Adds fields of another size parallely.
operator-= - Subtracts fields of another size parallely.
operator== - Compares two sizes for equality.
operator!= - Compares two sizes for inequality.

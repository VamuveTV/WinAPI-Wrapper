WinAPI Wrapper

HotKey

The HotKey class extends the HotKey standard Windows control with C++ functionality. It is derived from the Control class. See Hot Key Controls Overview for details.

The HotKey class has the following functions:

Function Related message Description
HotKey - Creates the object optionally assigning an existing control handle to it.
Create CreateWindowEx Creates the control.
GetHotKey HKM_GETHOTKEY Returns virtual key code.
GetHotKeyModifier HKM_GETHOTKEY Returns modifier flags.
SetHotKey HKM_SETHOTKEY Sets hotkey virtual key code and modifier flags.
SetRules HKM_SETRULES Defines invalid key combinations.

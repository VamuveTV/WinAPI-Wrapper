GDIObjectBase

The GDIObjectBase class is an autodestructible version of GDIObject.

The GDIObjectBase class has the following functions:

Function Related WinAPI function Description
GDIObjectBase - Creates an empty instance without any valid object assigned.
~GDIObjectBase DeleteObject Destroys the assigned GDI object.
Set DeleteObject Sets another GDI object destroying the current one.

WinAPI Wrapper

ComboBox

The ComboBox class extends the ComboBox standard Windows control with C++ functionality. It is derived from the Control class. See Combo Boxes Overview for details.

The ComboBox class has the following functions:

Function Related message Description
ComboBox - Creates the object optionally assigning an existing control handle to it.
Create CreateWindowEx Creates the control.
AddString CB_ADDSTRING Adds a string to the combo box list.
InsertString CB_INSERTSTRING Inserts a string into the combo box list.
DeleteString CB_DELETESTRING Deletes a string from the combo box list.
ResetContent CB_RESETCONTENT Clears the combo box list.
FindString CB_FINDSTRING Returns index of a strings that starts with the specified portion.
FindStringExact CB_FINDSTRINGEXACT Returns index of a specific string.
Dir CB_DIR Fills the combo box list with contents of a specific directory.
GetCount CB_GETCOUNT Returns the number of items in the combo box list.
GetCurSel CB_GETCURSEL Returns the index of the currently selected item in the combo box list.
SetCurSel CB_SETCURSEL Selects a specific item in the combo box list.
SelectString CB_SELECTSTRING Selects a specific string in the combo box list.
LimitText CB_LIMITTEXT Limits the allowable length of the user-entered string.
GetDroppedWidth CB_GETDROPPEDWIDTH Returns the minimum allowable width of the combo box's listbox.
GetDroppedState CB_GETDROPPEDSTATE Determines whether the combo box is dropped.
GetDroppedControlRect CB_GETDROPPEDCONTROLRECT Returns screen coordinates of the combo box in dropped state.
SetDroppedWidth CB_SETDROPPEDWIDTH Sets the minimum allowable width of the combo box's listbox in a dropped state.
GetEditSel CB_GETEDITSEL Gets starting and ending string index of the current selection in the edit box.
SetEditSel CB_SETEDITSEL Selects a range of text in the edit box.
GetExtendedUI CB_GETEXTENDEDUI Checks which combo box UI is currently used (default or extended).
SetExtendedUI CB_SETEXTENDEDUI Sets the combo box UI (default or extended).
GetHorizontalExtent CB_GETHORIZONTALEXTENT Returns the amount by which the list box can be scrolled horizontally.
SetHorizontalExtent CB_SETHORIZONTALEXTENT Sets the amount by which the list box can be scrolled horizontally.
GetItemData CB_GETITEMDATA Returns item-specific user data.
SetItemData CB_SETITEMDATA Sets item-specific user data.
GetItemHeight CB_GETITEMHEIGHT Returns item height.
SetItemHeight CB_SETITEMHEIGHT Sets item height.
GetLBTextLength CB_GETLBTEXTLEN Returns length of a string in the list box.
GetLBText CB_GETLBTEXT Returns string of a list box item.
GetLocale CB_GETLOCALE Returns current locale.
SetLocale CB_SETLOCALE Sets locale.
GetTopIndex CB_GETTOPINDEX Returns index of the first visible list box item.
SetTopIndex CB_SETTOPINDEX Makes a list box item the first displayed item.
ShowDropDown CB_SHOWDROPDOWN Shows or hides the dropping part of the combo box (the listbox).
GetComboBoxInfo CB_GETCOMBOBOXINFO Returns combo box information in form of the COMBOBOXINFO structure.

WinAPI Wrapper

GetFolderDialog

The GetFolderDialog class is an extension of the BROWSEINFO structure and it is derived from the structure. The common dialog opened by the class makes the user select a folder.

The GetFolderDialog class has the following functions:

Function Related WinAPI function Description
GetFolderDialog - Initializes the BROWSEINFO structure.
Go SHBrowseForFolder Opens the "Browse for folder" common dialog.
GetFolderName - Gets path to folder after the Go() function returned true.
GetIDList - Gets IDList returned by SHBrowseForFolder.

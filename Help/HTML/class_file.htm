WinAPI Wrapper

File

The File class extends HANDLE objects that are handles to files. See File I/O Overview for the details.

The File class has the following functions:

Function Related WinAPI function Description
File CreateFile Assigns INVALID_HANDLE_VALUE or calls SetTo().
~File CloseHandle Calls Close()
operator HANDLE - Returns the file handle.
SetTo CreateFile Opens or creates file.
IsOpen - Validates the handle.
Close CloseHandle Closes file.
Read ReadFile Reads a buffer from file.
Write WriteFile Writes a buffer to file.
GetSize GetFileSize Returns size of the file.
Seek SetFilePointer Moves file pointer within the file.
Truncate SetEndOfFile Makes the current position of the file pointer the end of file.
Flush FlushFileBuffers Saves all unsaved internal Windows file buffers to disk.

WinAPI Wrapper

ComboBoxEx

The ComboBoxEx class extends the ComboBoxEx standard Windows control with C++ functionality. It is derived from the Control class. See Extended Combo Boxes Overview for details.

The ComboBoxEx class has the following functions:

Function Related message Description
ComboBoxEx - Creates the object optionally assigning an existing control handle to it.
Create CreateWindowEx Creates the control.
DeleteItem CBEM_DELETEITEM Deletes an item.
GetComboControl CBEM_GETCOMBOCONTROL Returns handle of the child combobox.
GetEditControl CBEM_GETEDITCONTROL Returns handle of the child edit control.
GetExtendedStyle CBEM_GETEXTENDEDSTYLE Returns extended style.
GetImageList CBEM_GETIMAGELIST Returns image list.
GetItem CBEM_GETITEM Returns item configuration.
GetItemText CBEM_GETITEM Returns item text.
GetItemImage CBEM_GETITEM Returns item image.
GetItemParam CBEM_GETITEM Returns item user parameter.
GetUnicodeFormat CBEM_GETUNICODEFORMAT Returns unicode format setting.
HasEditChanged CBEM_HASEDITCHANGED Checks whether the contents of the edit control have changes.
InsertItem CBEM_INSERTITEM Inserts a new item.
SetExtendedStyle CBEM_SETEXTENDEDSTYLE Sets extended style.
SetImageList CBEM_SETIMAGELIST Sets image list.
SetItem CBEM_INSERTITEM Sets item configuration.
SetItemParam CBEM_INSERTITEM Sets item user parameter.
SetUnicodeFormat CBEM_SETUNICODEFORMAT Sets unicode format setting.

WinAPI Wrapper

Button

The Button class extends the Button standard Windows control with C++ functionality. It is derived from the Control class. See Buttons Overview for details.

The Button class has the following functions:

Function Related message Description
Button - Creates the object optionally assigning an existing control handle to it.
Create CreateWindowEx Creates the control.
Click BM_CLICK Simulates a user click.
GetCheck BM_GETCHECK Retrieves button check state.
SetCheck BM_SETCHECK Sets button check state.
GetState BM_GETSTATE Retrieves generic button state.
SetState BM_SETSTATE Sets generic button state.
SetStyle BM_SETSTYLE Changes button style.
GetBitmap BM_GETIMAGE Retrieves HBITMAP associated with the button.
GetIcon BM_GETIMAGE Retrieves HICON associated with the button.
SetBitmap BM_SETIMAGE Sets HBITMAP to the button.
SetIcon BM_SETIMAGE Sets HICON to the button.

WinAPI Wrapper

TreeItem

The TreeItem class extends the HTREEITEM tree item handle with C++ functionality. It is typically created by functions of the TreeView class.

The TreeItem class has the following functions:

Function Related macro Description
TreeItem - Constructs an object (either empty or from an existing handle).
operator HTREEITEM - Returns the tree item handle.
CreateDragImage TreeView_CreateDragImage Creates a dragging bitmap.
Delete TreeView_DeleteItem Removes the and all its children.
EditLabel TreeView_EditLabel Initiates editing of the item label (editing by the user).
EnsureVisible TreeView_EnsureVisible Ensures an item is visible.
Expand TreeView_Expand Expands the list.
ExpandPartial TreeView_Expand Partially expands the list.
Collapse TreeView_Expand Collapses the list.
CollapseReset TreeView_Expand Collapses the list and removes child items.
Toggle TreeView_Expand Toggles between collapsed and expanded list.
GetChild TreeView_GetNextItem Returns the first child item.
GetNextSibling TreeView_GetNextItem Returns next sibling item.
GetPrevSibling TreeView_GetNextItem Returns previous sibling item.
GetNextVisible TreeView_GetNextItem Returns next visible item.
GetPrevVisible TreeView_GetNextItem Returns previous visible item.
GetParent TreeView_GetNextItem Returns parent item.
GetItem TreeView_GetItem Returns the item configuration in form of TVITEM structure.
GetItemText TreeView_GetItem TreeView_GetItem
GetItemImage TreeView_GetItem Returns item image index.
GetItemParam TreeView_GetItem Returns item user parameter.
GetItemState TreeView_GetItem Returns current item state.
GetItemRect TreeView_GetItemRect Returns rectangle that contains the item.
InsertItem TreeView_InsertItem Inserts a new item.
InsertItemFirst TreeView_InsertItem Inserts a new item at the beginning of the list.
InsertItemLast TreeView_InsertItem Inserts a new item at the end of the list.
InsertItemSort TreeView_InsertItem Inserts a new item in alphabetical order.
Select TreeView_Select Selects the item making it visible (and can do other things as well).
SelectDropTarget TreeView_SelectDropTarget Redraws the item making it look like a drop target.
SelectItem TreeView_SelectItem Selects the specified item.
SelectSetFirstVisible TreeView_SelectSetFirstVisible Ensures that an item is visible. (See Select, SelectItem and EnsureVisible).
SetInsertMark TreeView_SetInsertMark Sets the insertion mark.
SetItem TreeView_SetItem Sets item configuration using the TVITEM structure.
SetItemState TreeView_SetItem Sets item state.
SetItemParam TreeView_SetItem Sets item user parameter.
SortChildren TreeView_SortChildren Sorts children alphabetically or using an user-specified compare function.

Remarks

There is some similarity between the Select, SelectItem, EnsureVisible and SelectSetFirstVisible functions.

WinAPI Wrapper

PaintDC

The PaintDC class is used to operate on a DC during window repainting operation, that is during processing the WM_PAINT message. It is derived from the DC class and has all its functionality. See DC class for details.

The PaintDC class has the following functions:

Function Related WinAPI function Description
PaintDC BeginPaint Obtains the DC for drawing.
~PaintDC EndPaint Releases the DC after drawing with it.
RequiresErase PAINTSTRUCT Checks whether the DC requires erasing (PAINTSTRUCT.fErase is true).
PaintRect PAINTSTRUCT Returns the update rectangle that has to be repainted (PAINTSTRUCT.rcPaint).

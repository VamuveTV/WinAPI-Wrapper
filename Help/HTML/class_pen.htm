WinAPI Wrapper

Pen

The Pen class provides automatic management (creating and destroying) of pen GDI objects. It is derived from the GDIObjectBase class. See Pens Overview for details.

The Pen class has the following functions:

Function Related WinAPI function Description
Pen CreatePen Creates a pen object.
operator HPEN - Returns pen handle.

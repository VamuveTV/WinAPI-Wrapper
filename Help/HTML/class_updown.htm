WinAPI Wrapper

UpDown

The UpDown class extends the UpDown standard Windows control with C++ functionality. It is derived from the Control class. See UpDown Controls Overview for details.

The UpDown class has the following functions:

Function Related message Description
UpDown - Creates the object optionally assigning an existing control handle to it.
Create CreateWindowEx Creates the control.
GetAccel UDM_GETACCEL Returns acceleration information.
GetBase UDM_GETBASE Returns current radix base (10 or 16).
GetBuddy UDM_GETBUDDY Returns buddy window.
GetPos UDM_GETPOS32 Returns current value.
GetRange UDM_GETRANGE32 Returns range of valid values.
SetAccel UDM_SETACCEL Sets acceleration information.
SetBase UDM_SETBASE Sets radix base (10 or 16).
SetBuddy UDM_SETBUDDY Sets buddy window.
SetPos UDM_SETPOS32 Sets new value.
SetRange UDM_SETRANGE32 Sets range of valid values.

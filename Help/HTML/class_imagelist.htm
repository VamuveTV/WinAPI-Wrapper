WinAPI Wrapper

ImageList

The ImageList class is a wrapper for HIMAGELIST objects. This is a non-destructive class - it does not destroy the image list object. For a destructive version see AutoImageList. See Image Lists Overview for details.

The ImageList class has the following functions:

Function Related WinAPI function Description
ImageList - Assigns an empty or an existing image list handle.
operator HIMAGELIST - Returns image list handle.
Create ImageList_Create Creates an image list.
Duplicate ImageList_Duplicate Makes a copy of the image list.
Destroy ImageList_Destroy Destroys image list.
LoadImage ImageList_LoadImage Loads an image from either a file or a resource.
Add ImageList_Add Adds a HBITMAP to the image list.
AddMasked ImageList_AddMasked Adds a HBITMAP to the image list and creates a mask for it.
AddIcon ImageList_AddIcon Adds a HICON to the image list.
Replace ImageList_Replace ImageList_ReplaceIcon Replaces an image with HBITMAP or HICON.
Remove ImageList_Remove Removes an image from image list.
SetIconSize ImageList_SetIconSize Sets image dimensions and removes all existing images.
GetIconSize ImageList_GetIconSize Obtains image size.
GetIcon ImageList_GetIcon Gets an image as an HICON.
Copy ImageList_Copy Makes a copy of an image.
Draw ImageList_Draw ImageList_DrawEx Draws an image on a HDC.
GetImageCount ImageList_GetImageCount Returns number of images in image list.
SetImageCount ImageList_SetImageCount Resizes image list.
GetImageInfo ImageList_GetImageInfo Retrieves IMAGEINFO structure.
Merge ImageList_Merge Merges two images.
SetOverlayImage ImageList_SetOverlayImage Selects overlay image and its mask.
BeginDrag ImageList_BeginDrag Begins image drag operation creating a drag image.
DragEnter ImageList_DragEnter Enters image drag operation mode locking all window drawing. Use this right after BeginDrag.
DragLeave ImageList_DragLeave Ends image drag operation mode unlocking all window drawing.
DragMove ImageList_DragMove Moves dragged image after mouse move.
DragShowNolock ImageList_DragShowNolock Shows or hides dragged image.
EndDrag ImageList_EndDrag Finishes drag operation by destroying the dragged image. Call this after DragLeave.
SetDragCursorImage ImageList_SetDragCursorImage Creates drag image combining the image with mouse cursor.
GetDragImage ImageList_GetDragImage Returns temporary image list used for dragging.
SetBkColor ImageList_SetBkColor Sets background color.
GetBkColor ImageList_GetBkColor Gets background color.

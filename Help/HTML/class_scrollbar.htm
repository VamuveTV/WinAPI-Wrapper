WinAPI Wrapper

ScrollBar

The ScrollBar class extends the ScrollBar standard Windows control with C++ functionality. It is derived from the Control class. See Scroll Bars Overview for details.

The ScrollBar class has the following functions:

Function Related message Description
ScrollBar - Creates the object optionally assigning an existing control handle to it.
Create CreateWindowEx Creates the control.
EnableArrows SBM_ENABLE_ARROWS Enables or disables both arrows on the scrollbar.
GetPos SBM_GETPOS Returns current scroll position.
SetPos SBM_SETPOS Sets scroll position.
GetRange SBM_GETRANGE Returns scroll range.
SetRange SBM_SETRANGE Sets scroll range.
SetRangeRedraw SBM_SETRANGEREDRAW Sets scroll range and redraws the scroll bar.
GetScrollInfo SBM_GETSCROLLINFO Returns scroll info in form of a SCROLLINFO structure.
SetScrollInfo SBM_SETSCROLLINFO Sets scroll info in form of a SCROLLINFO structure.

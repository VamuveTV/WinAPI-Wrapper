COMInitializer

The COMInitializer class simplifies initialization of the COM library. Creating an object of this class, either statically or in the main application function, you no longer have to remember to initialize or uninitialize the COM library.

The constructor of the COMInitializer class calls the CoInitialize(0) function and the destructor calls the CoUninitialize function.

The COMInitializer class is especially useful when using the IPtr<T> class which simplifies the use of COM.

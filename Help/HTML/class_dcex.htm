WinAPI Wrapper

DCEx

The DCEx class is used to operate on a window's DC with special capabilities (e.g. exclusively on non-client area, etc.). It is derived from the DC class and has all its functionality. See DC class for details.

The DCEx class has the following functions:

Function Related WinAPI function Description
DCEx GetDCEx Obtains the DC for drawing.
~DCEx ReleaseDC Releases the DC after drawing with it.

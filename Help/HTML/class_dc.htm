WinAPI Wrapper

DC

The DC class extends Device Context with C++ functionality. See Device Contexts Overview for details.

The DC class does not contain any create/delete functionality. This functionality is supplied by appropriate autodestructive classes:

Class Constructor calls: Destructor calls:
CompatibleDC CreateCompatibleDC DeleteDC
ClientDC GetDC ReleaseDC
WindowDC GetWindowDC ReleaseDC
DCEx GetDCEx ReleaseDC
PaintDC BeginPaint EndPaint

The DC class has the following functions:

StretchBlt StretchBlt Bit block transfer with stretching/skewing capability.

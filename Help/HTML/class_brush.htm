WinAPI Wrapper

Brush

The Brush class provides automatic management (creating and destroying) of brush GDI objects. It is derived from the GDIObjectBase class. See Brushes Overview for details.

The Brush class has the following functions:

Function Related WinAPI function Description
Brush CreateSolidBrush CreateHatchBrush CreatePatternBrush CreateBrushIndirect Creates a brush object.
operator HBRUSH - Returns brush handle.

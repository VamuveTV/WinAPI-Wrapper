MessageQueue

The MessageQueue class extends the MSG structure with C++ functionality. Is is provided primarily for internal use by the Application<T> class. See Messages and Message Queues Overview for details.

The MessageQueue class has the following functions:

Function Related WinAPI function Description
GetMessage GetMessage Gets a message from the actual queue to the MessageQueue object removing the message from the queue. If the message queue is empty, the function waits until some message appears in it.
PeekMessage PeekMessage Similar to GetMessage, but does not wait for messages if the message queue is empty.
TranslateMessage TranslateMessage Resolves key and system key messages after GetMessage or PeekMessage.
DispatchMessage DispatchMessage Dispatches message to a window procedure (the window is specified by the message itself).
TranslateAccelerator TranslateAccelerator Translates message through an accelerator table to create a WM_COMMAND message.
TranslateMDISysAccel TranslateMDISysAccel Similar to TranslateAccelerator, but works with a Windows internal accelerator table for MDI children.
IsDialogMessage IsDialogMessage Processes dialog box messages - useful for modeless dialog boxes.

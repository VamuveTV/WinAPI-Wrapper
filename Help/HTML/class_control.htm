WinAPI Wrapper

Control

The Control class is a base class for all control classes. It simplifies the creation of controls. It is derived from the GenericWindow class.

The Control class has the following functions:

Function Related WinAPI function Description
Control - Creates the control from a specified window handle (possibly an empty one).
Create CreateWindowEx Creates a new control.
Assign - Assigns another HWND control handle to the control object.

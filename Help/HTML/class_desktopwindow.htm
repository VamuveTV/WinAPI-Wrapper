DesktopWindow

The DesktopWindow class is an enhancement of the GenericWindow class. It initially contains the handle of the desktop window.

The DesktopWindow class has the following functions:

Function Related WinAPI function Description
DesktopWindow GetDesktopWindow Obtains handle of the desktop window.

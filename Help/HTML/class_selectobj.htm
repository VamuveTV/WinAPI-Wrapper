SelectObj

The SelectObj class simplifies the process of selecting and deselecting GDI objects to a DC. Using this class makes it easier to do the selection and deselection and reduces the number of bugs made during writing drawing code.

The SelectObj class has the following functions:

Function Related WinAPI function Description
SelectObj SelectObject Selects a GDI object to a particular DC.
~SelectObj SelectObject Deselects a GDI object from a DC.
Select SelectObject Selects another GDI object to the DC.
Deselect SelectObject Forces deselection of a GDI object from the DC.

Remarks

In general, the process of selecting and deselecting GDI objects to a DC can introduce bugs, while the appropriate selecting and deselecting code is separated by other code. Using the SelectObj class can reduce the number of bugs. The following example shows how to use it: 

{   SelectObj selector( dc, pen );
    dc.LineTo( p );
}
In the above code, as soon as the closing clause } is used, the pen is deselected from the DC by the selector's destructor. It is a safe way of selecting objects but it has one danger:
NOTE!
Always create the object of the class SelectObj because it is easy to mistake the class for the SelectObject function. If you don't specify object's name as in the following code, there will be created a temporary object that will die in the next line of your code, deselecting the GDI object back from the DC. 
// WARNING: wrong code
SelectObj( dc, pen );
